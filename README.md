# ClipDock

ClipDock is a local-first macOS productivity utility for saving and managing links, text snippets, prompts, and commands.

## Features

- Save four types of items: Link, Text, Prompt, and Command
- Search by title, content, or tag
- Sort by newest, oldest, and most used
- Open links and copy saved content quickly
- Trash and restore items within 10 days
- Quick Add floating window for lightweight capture
- Local-first storage using UserDefaults / JSON

## Tech Stack

SwiftUI, macOS App, Xcode, UserDefaults, JSON, Git
